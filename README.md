# ascii-snake
i've got a snake in my konsole

# Setup (linux)

## easy
run setup.sh in your konsole
## manual
`mkdir build` <br>
`cd build` <br>
`cmake ../` <br>
`cd ..` <br>

# Building (linux)

## easy
run build.sh in your konsole
## manual:
`cmake --build build` <br>
